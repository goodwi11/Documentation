===========
Инструменты
===========

.. note::
 Инструменты доступны для всех пользователей с активной подпиской **BASIC**, **PRO** или **BUSINESS**.

Данный раздел содержит набор утилит, упрощающих работу при создании и подготовке контента для арбитражных и маркетинговых задач.
Каждый инструмент выполняет свою функцию — от генерации данных до уникализации изображений и видео.

Уникализаторы
=============

Уникализаторы предназначены для изменения изображений и видео с целью повышения их уникальности.
Инструмент полезен для арбитражных кампаний, тестирования контента и обхода повторного детектирования медиа.

Уникализатор фото
-----------------

Позволяет загрузить изображение и преобразовать его до уникального состояния, сохраняя визуальное качество.
Доступно три степени уникализации: Слабая, Средняя, Сильная.

**Как использовать**

**Шаг 1.** Загрузите изображение (формат: JPG, PNG, WEBP и другие).

**Шаг 2.** Выберите степень уникализации.

**Шаг 3.** Нажмите **«Уникализировать»**.

**Шаг 4.** После обработки появится предпросмотр полученного изображения.

**Шаг 5.** Нажмите **«Скачать»** для получения готового изображения.

Генераторы данных
=================

Генераторы данных предназначены для создания текстовых данных, таких как адреса и номера телефонов, в соответствии с форматами выбранной страны.
Инструменты помогают быстро получать валидные данные для заполнения форм, лендингов и рекламных страниц.

Генератор адресов
-----------------

Позволяет получить случайно сгенерированный адрес по стандартам выбранной страны.

**Как использовать**

**Шаг 1.** Выберите страну из выпадающего списка.

**Шаг 2.** Нажмите **«Сгенерировать»**.

**Шаг 3.** В поле **«Готовый адрес»** появится результат.

**Шаг 4.** Нажмите **«Скопировать»**, чтобы сохранить адрес в буфере обмена.

Генератор телефонных номеров
----------------------------

Позволяет получить случайно сгенерированный телефонный номер в соответствии с кодом и структурой выбранной страны.

**Как использовать**

**Шаг 1.** Выберите страну из выпадающего списка.

**Шаг 2.** Нажмите **«Сгенерировать»**.

**Шаг 3.** В поле **«Готовый номер»** появится результат.

**Шаг 3.** Нажмите **«Скопировать»**, чтобы сохранить номер в буфере обмена.
